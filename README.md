# Dashboard Anual de Manifestações – Ouvidoria (Excel)

## Sobre o projeto

Este projeto consiste em um dashboard interativo desenvolvido em Microsoft Excel para análise de manifestações de uma Ouvidoria.

O objetivo foi transformar dados tabulares em informações úteis para acompanhamento de indicadores e apoio à tomada de decisão, utilizando recursos nativos do Excel.

> **Observação:** Os dados utilizados são fictícios e/ou anonimizados, criados exclusivamente para fins de estudo e composição de portfólio.

## Principais indicadores

- Total de manifestações
- Tempo médio de resposta
- Percentual respondido dentro do prazo
- Distribuição por tipo de manifestação
- Evolução mensal das manifestações
- Distribuição por unidade/setor
- Distribuição por UF

---

## Recursos utilizados

- Microsoft Excel
- Tabelas Dinâmicas (Pivot Tables)
- Gráficos Dinâmicos
- Segmentação de Dados (Slicers)
- KPIs
- Dashboard Executivo

## Como utilizar

1. Faça o download do arquivo `Dashboard.xlsx`.
2. Abra no Microsoft Excel.
3. Utilize os filtros (Slicers) para explorar os dados de forma interativa.
